# 🌀 AI Gesture-Controlled Portal System

An AI-powered Computer Vision project that creates an interactive magic portal using **Python**, **OpenCV**, **MediaPipe**, and **Pygame**. The application tracks hand gestures in real time and renders an animated portal with glowing effects, particle systems, gesture recognition, portal sound effects, screenshot capture, and video recording.

---

## 🚀 Features

- ✋ Real-Time Hand Tracking
- 🌀 AI Gesture Recognition
- 🌟 Animated Magic Portal
- 🔥 Dynamic Particle Effects
- 💫 Portal Glow Animation
- 🎯 Smooth Portal Movement
- 🖼️ Background Capture & Portal Reveal
- 🔊 Portal Sound Effects
- 📸 Screenshot Capture
- 🎥 Video Recording
- 📈 Live FPS Counter

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/ai-gesture-controlled-portal.git
```

```bash
cd ai-gesture-controlled-portal
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```
